A modern WordPress block theme with clean white space, mobile-first design, and Full Site Editing support.

== Description ==

Creatlach is a modern WordPress block theme built for Full Site Editing (FSE). It features a clean, minimal aesthetic with generous white space, a mobile-first responsive layout, and fluid typography that scales gracefully across all screen sizes.

Built entirely on the WordPress block editor and theme.json configuration, Creatlach requires no page builder and no custom build tools. Every aspect of the theme — from colours and spacing to typography and layout — is managed through global styles, making it straightforward to customise directly in the Site Editor.

Key features:

* Full Site Editing support with template and template part editing
* Fluid typography using clamp() for smooth scaling between breakpoints
* Comprehensive colour palette with primary, secondary, muted, and accent colours
* Fluid spacing scale for consistent rhythm throughout the design
* Mobile-first responsive design
* WCAG AA accessible colour combinations
* Block patterns covering hero sections, content layouts, calls to action, footers, and more
* No JavaScript dependencies for core theme functionality
* Clean, semantic HTML structure

== Installation ==

1. In your WordPress admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and choose the creatlach.zip file. Click Install Now.
3. Click Activate to use your new theme right away.

Alternatively, upload the theme folder directly to /wp-content/themes/ via FTP and activate it from the Themes screen.

== Frequently Asked Questions ==

= Does this theme support custom menus? =

Yes. Navigation menus are managed through the Navigation block in the Site Editor (Appearance > Editor).

= How do I change colours and fonts? =

Open the Site Editor (Appearance > Editor) and click Styles in the top right corner. All global style options are available there, based on the palette and typography defined in theme.json.

= Does this theme work with WooCommerce? =

The theme includes Commerce block patterns suitable for product displays, but it does not bundle WooCommerce-specific template overrides. WooCommerce compatibility may be added in a future release.

= Where are the block patterns? =

All custom patterns are available in the block inserter under the Patterns tab, organised into categories: Hero & Header, Content, Call to Action, Gallery & Media, Text & Typography, Footer, Blog & Archive, Commerce, and Utility.
